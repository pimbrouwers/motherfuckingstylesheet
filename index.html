Atomic Classes: Just Enough To Not Lose Your Shit

Look, sometimes you need a utility class. Not the whole fucking utility buffet that makes your HTML look like it was written by a cat walking across your keyboard:

<div class="flex flex-col md:flex-row justify-between items-center px-4 py-6 mx-auto rounded-lg shadow-md bg-white dark:bg-gray-800 hover:shadow-xl transition-all duration-300">Kill me now</div>

Seriously, what the fuck is that? A div, or a goddamn spaceship control panel?

Instead, this stylesheet gives you just enough utilities to handle edge cases without turning your markup into alphabet soup:

Each utility connects to the same variables as the rest of your stylesheet, so everything stays consistent. Your margins will match your paddings will match your borders. Like a civilized fucking design system should. 

.flex { display: flex }
.w-100 { width: 100% }
.center { margin-inline: auto }
.mt3 { margin-top: var(--s3) }
.mb0 { margin-bottom: 0 }
.pa2 { padding: var(--s2) }

Look at that. Minimal. Purposeful. Fucking useful.

Want to see all the atomic classes? They're at the bottom of the page. There aren't many because this isn't a bloated framework.

Install

You want to make a beautiful website? Good. But you don't want to spend hours tweaking pixels like a fool? Even better. Just include base.css like this:

<link rel="stylesheet" href="base.css">

Want to find base.css? Open DevTools (F12 like a pro), go to the Network tab, reload the page, and look for a file named base.css. go to the Network tab, reload the page, and look for something called base.css. If that sounds too complicated, just right-click anywhere, hit Inspect, and start clicking around like a caffeinated raccoon digging through a trash can. You'll find it. I believe in you.

Boom. That's it. Your site is now stunning. No 20MB framework, no 500-class utility system, no CSS-in-JS nonsense. Just one damn file that makes everything look good.
